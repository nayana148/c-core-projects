# C Core Projects

This repository contains core data structure and algorithm implementations
written in C to strengthen problem-solving and low-level programming skills.

## Topics Covered
- Stack implementation using arrays
- Queue implementation using arrays
- Singly linked list
- Circular linked list
- Searching algorithms (Binary Search)
- Sorting algorithms (Bubble Sort, Selection Sort)

## Why This Repository?
- Focuses on pointer manipulation and memory management
- Builds strong fundamentals required for technical interviews
- Forms the foundation for higher-level system and ML projects
